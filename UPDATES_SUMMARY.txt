🎯 DESIGN SYSTEM DETAILS
═══════════════════════════════════════════════════════════════════════════════

COLOR SCHEME (Dark Mode - Default):
  • Background:     oklch(0.12 0 0)  → Deep charcoal
  • Primary:        oklch(0.7 0.19 235) → Vibrant cyan-blue (PSX)
  • Secondary:      oklch(0.75 0.17 160) → Bright green (verified/success)
  • Card:           oklch(0.15 0 0)  → Elevated panels
  • Border:         oklch(0.25 0 0)  → Subtle dividers
  • Text Primary:   oklch(0.98 0 0)  → High contrast white
  • Text Secondary: oklch(0.65 0 0)  → Muted gray

ANIMATIONS (GPU-Accelerated, No Parallax):
  • Card hover:    translate-y(-2px to -8px) over 300ms
  • Icon hover:    scale(110%) over 200ms
  • Button arrow:  translate-x(+4px) over 200ms
  • Background:    Pulse animation (1s-2s delay between orbs)

SPACING & SIZING:
  • Border radius: rounded-xl (12px for icons), rounded-lg (8px for cards)
  • Gaps: Responsive (grid-cols-1 → 2 → 3 based on screen size)
  • Padding: p-6 to p-12 depending on component
